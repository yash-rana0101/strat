## SDKs & Helper Clients

Speed up your integration using our pre-built software development kits.

### Available Packages
- **Python SDK**: `pip install strat-ai` — Ideal for backtesting pipelines and data scientists.
- **Node.js Client**: `npm install @strat-ai/client` — Perfect for web developers and dashboard integrations.
- **Go Client**: `go get github.com/stratai/client-go` — Designed for high-concurrency systems.
